# haBITS - A Modern Habit and Task Tracker

## Overview

haBITS is a modern and intuitive web application designed to help users track their daily habits and tasks. It provides a clean and user-friendly interface that allows users to easily add, complete, and manage their habits and tasks. The application is built with a focus on user experience, with a visually appealing design and interactive elements that provide clear feedback to the user.

## Project Outline

**Core Features:**

*   **Habit Tracking:** Users can create, view, and complete daily habits.
*   **Task Management:** Users can create, view, and complete daily tasks.
*   **Intuitive UI:** A clean and modern user interface that is easy to navigate.
*   **Visual Feedback:** Interactive elements with glowing effects and multi-layered drop shadows.
*   **Modern Design:** A visually balanced layout with clean spacing and polished styles.
*   **Responsive Design:** The application is mobile responsive and adapts to different screen sizes.
*   **Iconography:** Icons are used to enhance the userâ€™s understanding and the logical navigation of the app.

**Technology Stack:**

*   **Frontend:** Next.js, React
*   **Styling:** CSS Modules, with a focus on modern design principles.
*   **Icons:** react-icons

**Project Structure:**

*   `pages/`: Contains the main pages of the application (`index.js`, `habits.js`, `tasks.js`, `_app.js`).
*   `components/`: Contains reusable React components (`Layout.js`, `Navbar.js`, `CreateHabitModal.js`, `CreateTaskModal.js`).
*   `styles/`: Contains CSS Modules for styling the application (`globals.css`, `Habits.module.css`, `Tasks.module.css`, `Navbar.module.css`, `CreateHabitModal.module.css`, `CreateTaskModal.module.css`).
*   `public/`: Contains static assets, such as images and fonts.
